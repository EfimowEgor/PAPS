# Разработка системы для геймификации процесса изучения математики. 

## Пользователи: Студенты (ученики школ, студенты ВУЗов), Преподаватели, Независимые пользователи (те кто решают самостоятельно)

## Описание
Российская математическая школа за исключение пары физмат лицеев и ВУЗов переживает тяжёлое время. Людям неинтересно изучать математику, неинтересно решать задачи. Материал изучается очень неглубоко. Существуют решения, например, Khan Academy, где есть возможность решать маленькие задачи и смотреть короткие видео. Однако, проблема заключается в том, что недостаточное количество практики, а в занятиях не уделяется внимание деталям. Нет возможностей организовывать студенческие группы. Геймификация системы существует только для вида, она не открывает никаких возможностей.   

## Требования

* Регистрация и авторизация пользователей.
* Система очков, уровней и наград за выполнение заданий.
* Геймификация заданий.
* Создание и редактирование курсов.
* Организация студенческих групп.
* Выгрузка статистики по студентам.
* Генерация заданий.
* Реализация рейтингового режима.
